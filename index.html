Choose Our Own Adventure

COOA is a web-based microframework that makes it possible to collaboratively author and play hypertext narratives.

It is optimized for development inside TogetherJS-enabled editors like Thimble and JSFiddle. However, it can also be developed within JS Bin or your favorite text editor.

General Usage

  1. Drop cooa.css and cooa.js into an HTML page.
  2. Create a div, or any other container element of your choice, inside your page and give it the class cooa.
  3. Add child <section> elements to this container. These will hold the individual “pages” of your story. Give each section a unique id attribute. By default, only the first section will be shown to the user; the others will be hidden.
  4. Link between different sections using fragment identifiers. For instance, <a href="#foo"> will take a user to <section id="foo"> by hiding the current section and showing the foo section.

See the example for more information.

Debug Mode

If you add a checkbox with id cooa-debug anywhere on your HTML page, enabling it activates debug mode.

In debug mode, all sections of your story are shown at once, and the currently active section is highlighted.

TogetherJS Integration

If you want to play a story collaboratively via TogetherJS, you can do so by following the standard instructions for integration and loading cooa.together.js immediately after togetherjs-min.js. See the example for more details.
